About MORPH

Technical Overview

What is MORPH

MORPH (Modular Reactive Polymorphic Honeypot) is an adaptive SSH honeypot system built on Cowrie. Unlike static honeypots that are easily fingerprinted, MORPH actively responds to attacker behavior - planting new deception artifacts mid-session and evolving its environment across sessions based on attack history.

How It Works

Attacker -> Cowrie SSH -> Log Parser -> Classifier -> Dossier
Reactor (real-time) -> honeyfs modification
Adaptor (cross-session) -> environment evolution

Key Capabilities

Real-time deception: files planted based on attacker commands
Behavioral classification: bot/human, intent, risk (rule-based)
Cross-session memory: environment adapts from attack history
Attacker dossiers: structured JSON per session
OSINT enrichment: IP geolocation and ASN via ipinfo.io
Live dashboard: Flask + HTMX, no JS frameworks
